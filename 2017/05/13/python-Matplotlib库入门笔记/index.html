pyplot的plot()函数

  • x: X轴数据，列表或数组，可选
  • y: Y轴数据，列表或数组
  • format_string: 控制曲线的格式字符串，可选
  • **kwargs: 第二组或更多(x,y,format_string)

当绘制多条曲线时，各条曲线的x不能省略



format_string: 控制曲线的格式字符串，可选 由颜色字符、风格字符和标记字符组成

颜色字符 说明 颜色字符 说明
‘b’ 蓝色 ‘m’ 洋红色 magenta
‘g’ 绿色 ‘y’ 黄色
‘r’ 红色 ‘k’ 黑色
‘c’ 青绿色cyan ‘w’ 白色
‘#008000’ RGB某颜色 ‘0.8’ 灰度字符串
风格字符 说明
‘-‘ 实现
‘–’ 破折线
‘-.’ 点画线
‘:’ 虚线
‘’ ‘’ 无线条
